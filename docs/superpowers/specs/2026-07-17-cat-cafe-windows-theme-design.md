# 一杯猫 · 暖阳咖啡店 Windows 主题设计

## 目标

为 Codex Dream Skin 增加一套简单、低干扰的 Windows 浅色主题预设。主题以用户提供的猫咪咖啡店插画为唯一背景素材，在保留原画构图的同时，通过暖色半透明界面层保证 Codex 原生控件和文字清晰可读。

## 范围

- 平台：Windows。
- 外观：仅针对 Codex 浅色模式优化。
- 交付形态：可由 Windows 本地主题仓库保存、切换和恢复的主题预设。
- 不修改 Codex 官方安装包、WindowsApps、`app.asar`、API 配置或模型供应商设置。
- 不新增动画、声音、独立设置页面或额外运行时依赖。

## 视觉设计

主题名称为“一杯猫 · 暖阳咖啡店”。背景使用用户提供的 `2912 × 1632` PNG，实际比例为 91:51（约 16:9）。保持画面中的猫、咖啡设备、植物、菜单板和柜台完整可见，不把界面元素直接烘焙进图片，也不对原图做生成式重绘。

基础色板：

- 主色：奶油黄 `#FFE8B0`。
- 辅色：焦糖棕 `#9B6B52`。
- 强调色：蜜桃橙 `#E99A72`。
- 正文色：深咖啡色，须满足浅色界面中的清晰阅读需求。

界面层：

- 侧栏使用约 76% 不透明度的暖白底色。
- 建议卡片和项目卡片使用约 68% 不透明度的暖白底色、轻微背景模糊及浅棕描边。
- 输入框使用约 88% 不透明度的暖白底色，优先保障输入文字、占位文本和按钮的可读性。
- 首页保留较完整的插画氛围；进入任务页后降低背景存在感，避免影响长时间阅读和编码。
- 焦点态、选中态和主要操作使用蜜桃橙或焦糖棕，不使用高饱和冷色。

具体透明度可在实机验证时小幅调整，但不得改变“暖色、浅色、低干扰”的设计方向。

## 主题结构与数据流

预设遵循现有 Windows 主题保存和切换结构，不引入新的主题格式。背景图作为主题资产随预设存放；主题配置记录显示名称、背景资产引用、浅色调色参数和界面覆盖参数。

切换流程保持现状：用户从 Windows 托盘或已保存主题列表选择预设，现有脚本读取主题配置并通过仅绑定 `127.0.0.1` 的 CDP 会话注入背景与样式。恢复操作继续使用项目已有恢复路径，主题不得影响官方文件或其他 Codex 配置。

## 自适应与降级

- 背景采用覆盖式 16:9 显示，并以画面中央区域为默认构图基准。
- 在非 16:9 窗口中允许边缘裁切，但优先保留中央菜单板、柜台猫咪和右侧咖啡设备。
- 当背景模糊不可用时，半透明底色仍须提供足够对比度。
- 如果主题资产缺失或不可读，沿用现有错误处理和恢复机制，不留下半应用状态。

## 验收标准

1. 预设可在 Windows 已保存主题列表中识别并切换。
2. Codex 浅色首页显示完整、协调的猫咪咖啡店主题，原生控件仍可交互。
3. 侧栏、建议卡片、项目选择区和输入框文字清晰，无明显重影或低对比度问题。
4. 任务页背景明显弱于首页，代码、对话和状态文字易读。
5. 常见 16:9 窗口下不裁掉主题主体；窄窗口下保持主要视觉元素可辨认。
6. Windows 回归测试通过，并完成首页与任务页的实机浅色外观检查。
7. 暂停、切换其他主题和恢复官方外观均正常。
8. CDP 仍只绑定回环地址，且不修改官方安装目录、API Key、Base URL 或其他无关配置。

## 测试计划

- 运行 `powershell -File windows/tests/run-tests.ps1`。
- 验证安装、启动、主题切换、暂停和恢复路径。
- 在 Codex 浅色模式下检查首页及任务页，并记录视觉结果。
- 检查主题名称和路径中的中文字符能被正确保存与读取。
- 检查背景资产缺失时的失败与恢复行为。
